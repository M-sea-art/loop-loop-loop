# Observation Engine

Inspect the final artifact in the form in which the user experiences it.

- Visual output must be rendered and viewed across contract scenarios.
- Code must be exercised through observable behavior, not merely parsed or built.
- Data must be recomputed against raw inputs and edge cases.
- Documents must be rendered, paginated, and inspected.
- Automation must verify the post-action environment state.
- Research must trace claims to current primary sources.

An execution narrative is not observation. A generated artifact is not an inspected artifact.
