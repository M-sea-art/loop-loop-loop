# Evidence Hardening

## 中文

LoopLoopLoop 不应该因为文件存在、代码编译成功或 Agent 自己声明完成，就认为目标已经完成。

可靠证据必须绑定：

- Goal
- Outcome
- Scenario
- Artifact
- Verification

目标是证明现实结果，而不是证明执行活动发生。

## English

LoopLoopLoop should not consider a goal complete because a file exists, code compiles, or an agent claims success.

Reliable evidence must connect:

- Goal
- Outcome
- Scenario
- Artifact
- Verification

The purpose is to prove real outcomes, not activity.
